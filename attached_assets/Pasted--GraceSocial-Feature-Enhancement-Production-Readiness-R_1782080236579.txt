30. Improve media interaction behavior on the Home Feed:

### Video Posts

* When a user taps on a video in the Home Feed, it should open in the dedicated Realms viewer similar to Instagram Reels.
* Open in full-screen mode.
* Users should be able to:

  * Like
  * Comment
  * Share
  * Save
  * Repost
  * Follow creator
  * View creator profile
  * Scroll to next and previous Realms
* Video playback should continue seamlessly when opened from the Home Feed.

### Photo Posts

* When a user taps on a photo post in the Home Feed, it should open the post details page with comments visible.
* Users should be able to:

  * View the image in full size
  * Read comments
  * Add comments
  * Reply to comments
  * Like comments
  * Share posts
  * Save posts
  * Repost posts

### Multi-Image Carousel Posts

* Tapping a carousel should open a detailed post view.
* Users should be able to:

  * Swipe through all images
  * Access comments from the same screen
  * Zoom in and out of images
  * Like, share, save, and repost

### User Experience Requirements

* Navigation should closely match Instagram.
* Opening media should feel instant with smooth animations.
* Returning to the Home Feed should restore the previous scroll position.
* Comments should load dynamically without refreshing the page.
* All interactions should update in real time.
